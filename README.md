# Anime-Data-Analysis
This project analyzes an Anime dataset to understand viewer preferences, rating distribution, and genre popularity.

## Project Overview

This project focuses on analyzing an Anime dataset to understand viewer preferences, rating patterns, genre popularity, and anime format trends.
The main goal is to extract meaningful insights using data analysis and visualization techniques.

By exploring different features such as rating, number of episodes, members, genre, and type, this project helps identify factors that influence anime popularity and viewer engagement.

---

## Objectives

* Analyze the distribution of anime ratings
* Identify the most popular anime genres
* Understand the relationship between episodes, ratings, and members
* Compare popularity across different anime formats such as TV, Movie, OVA, and ONA
* Generate insights using visualizations to support data-driven understanding

---

## Dataset Features

* **anime_id** – Unique ID of each anime
* **name** – Title of the anime
* **genre** – Category such as Action, Comedy, Drama, Fantasy, etc.
* **type** – Format (TV, Movie, OVA, ONA, Special, Music)
* **episodes** – Total number of episodes
* **rating** – Average viewer rating (1–10 scale)
* **members** – Number of users who added the anime to their watchlist
